## 2. Conteúdos Aproveitáveis

### 2.1. Projeto Coletor de Humidade — ANDRÉ BLANCO (Máxima relevância) 🔴

**Este é o único material que é PROPRIEDADE do grupo e pode ser usado como portfólio.**

| Característica | Detalhe |
|---|---|
| **Autores** | André Blanco (Labiapa) + Gianluca Stasi |
| **Local** | Escola de Engenharia Central, Piracicaba/SP |
| **Conceito** | Torre coletora de umidade atmosférica em treliça |
| **Materiais** | Bambu/aço + malha de condensação + tanque coletor |
| **Filosofia** | Autoconstrução com materiais locais |
| **Público-alvo** | Comunidades carentes, áreas de déficit hídrico |

**Uso na proposta Vaga Lúmen:**
- **Portfólio técnico** do grupo — comprova capacidade executiva
- **Estado da arte próprio** — o grupo já desenvolveu solução similar
- **Adaptação** — o conceito da torre fixa pode ser adaptado para o módulo itinerante (ônibus + barco)

### 2.2. Warka Water (2012) — Alta relevância 🟢

- Estrutura de **bambu** de 10m, 80kg
- Montagem: 10 pessoas em 30 minutos
- Prova de que **bambu + malha tensionada** funciona para captação de água
- **Referência** para justificativa de viabilidade técnica

### 2.3. Cecelic Atrapanieblas (2011) — Alta relevância 🟢

- Produção: 40 litros/dia em comunidades chilenas
- Materiais simples: malha PVC, polietileno, suporte metálico
- Modelo de baixo custo para **Linha 3 — Água e Esgoto** da FINEP

### 2.4. NRP 3.0 / Coastal Fog Tower / DropNet — Média relevância 🟡

- Bom para **estado da arte** na Justificativa de Incerteza
- Mostra que a tecnologia não é ficção — há precedentes globais
- NÃO usar como portfólio próprio
